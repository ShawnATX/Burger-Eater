
# Burger Eater

## Description

![Repo commit](https://img.shields.io/github/last-commit/ShawnATX/Burger-Eater)

This is a full stack application utilizing a SQL database, node/express server, and bootstrap styling on the front end. The app lets users create new 'burgers' which can then be eaten. The deployed application can be found at https://dry-woodland-14799.herokuapp.com/index"

## Installation

`npm i` in the root package directory to install dependencies.

## Usage

The application can be started by first using the schema.sql and seed.sql files to initialize a local MySQL database and load starter data. From there ensure your local connection information is saved in the connection.js file. You will need to add a `.env` file in the root directory to store your password, this will be saved on a single line in the format `MY_PASS=<your password>`
